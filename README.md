# SauceDemo Automation Test

This project contains automated test scripts for https://www.saucedemo.com/
using Selenium WebDriver with Java.
- Java
- Selenium WebDriver
- JUnit 5
- Maven
- WebDriverManager

Test Coverage
1. Login Test
- Valid and invalid users
- Error message validation
- URL validation

2. Add to Cart
- Add product to cart
- Verify product name
- Verify quantity

3. Checkout Flow
- Fill user information (random data)
- Complete checkout
- Validate success message
